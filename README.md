<h1> Projeto Amigo Secreto 🎁</h1>

<h2>🔍 Sobre</h2>
<p>Este é um simples programa para sortear amigos secretos.</p>

<h3>📌 Como usar?</h3>
<p>1- Adicione os nomes dos participantes.</p>
<p>2- Clique em "Sortear Amigo".</p>
<p>3- Nome sorteado será exibido na tela.</p>
